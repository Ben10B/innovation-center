Here, you'll marvel at my transitions. Don't worry. I'll add more in the future. This is all I've got! FYI, you have to hover to see them.

Extend

If you want to extend the width or height, add the extnd-'property'-0s class. You can change the duration of the transition up to 60s.

Example
                    
                        <div class="bg-second extnd-width-4s txt-center fnt-white"><h1>1</h1></div>
                        
1

                        <div class="bg-second extnd-height-1s txt-center fnt-white"><h1>2</h1></div>
                        
2

Change Color

If you want to change the color, add the changeTo-'color'-0s class. You can change the duration of the transition up to 60s.

Example
                    
                        <div class="bg-primary changeTo-second-2s txt-center fnt-white"><h1>1</h1></div>
                        
1

                        <div class="bg-second changeTo-third-2s txt-center fnt-white"><h1>2</h1></div>
                        
2

                        <div class="bg-third changeTo-white-2s txt-center fnt-primary"><h1>3</h1></div>
                        
3

Extend & Change Color

If you want to extend AND change the color, add the extnd-'property'-'color'-0s class. You can change the duration of the transition up to 60s.

Example
                    
                        <div class="bg-primary extnd-width-second-2s txt-center fnt-white"><h1>1</h1></div>
                        
1

                        <div class="bg-third extnd-height-white-2s txt-center fnt-primary"><h1>2</h1></div>
                        
2
