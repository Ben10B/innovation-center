.styling { HTML }

Here, you'll learn how my styling works. Don't worry. I'll add more in the future. This is all I've got!

Color

I use 3 colors for my theme. With these classes, you can change the background color as well as the font color.

Background Colors

To change the background color, use the bg-'color' class.

Example
                            
                                <p class="bg-primary">Primary Color</p>
                                
Primary Color

                                <p class="bg-second">Secondary Color</p>
                                
Secondary Color

                                <p class="bg-third">Third Color</p>
                                
Third Color

Light Background Colors

To change the background color, use the bg-'color'-light class.

Example
                            
                                <p class="bg-primary-light">Primary Color</p>
                                
Light Primary Color

                                <p class="bg-second-light">Secondary Color</p>
                                
Light Secondary Color

                                <p class="bg-third-light">Third Color</p>
                                
Light Third Color

Containers

I have 2 containers. They could use more work, yet they work.

Row

Content in rows should flow from left > right, right? To create a row container, use the row class.

Example
                            
                                <div class="row">
                                    <p class="bg-second txt-center">Left</p>
                                    <p class="txt-center">Center</p>
                                    <p class="bg-second txt-center">Right</p>
                                </div>

Column

Content in columns should flow from top > bottom, right? To create a column container, use the column class.

Example

Buttons

I like my buttons. You do too. 4 variations! To style a button, use the btn class. If you want a background color, add the bg-'color'-hvr class.

Example
                            
                                <button type="button" class="btn">Button</button>
                                
                                <button type="button" class="btn bg-primary-hvr">Button</button>
                                
                                <button type="button" class="btn bg-second-hvr">Button</button>
                                
                                <button type="button" class="btn bg-third-hvr">Button</button>

Sizing Dimensions

With these classes, you can adjust the width/height with ease! To adjust the width, use the wdth-'number' class. If you want to adjust the height, add the hght-'number' class.

Example
                            
                                <div class="bg-primary hght-10 wdth-50">This could be content.</div>
